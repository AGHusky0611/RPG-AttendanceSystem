# RPG-AttendanceSystem

This system will serve as the Attendance System for the Rated:Production Guild

Features

Admin side: 
- The admin should be able to change the directory of Excel sheet to read
- The admin should be able to upload or change the directory of the background photo

System Side:
- The system should be able to read the slu id of the students
- The system should be able to see if they have registered online or not

User side:
- The user should be able to input their id
- The user should be able to mark whether someone is present
- The user should be able to see their names before marking as present
